# Документация по UI/UX-дизайну

## Философия дизайна

### Основные принципы
- **Ясность превыше всего**: Каждый элемент интерфейса должен быть сразу понятен
- **Оптимизация под касание**: Все взаимодействия разработаны для сенсорных экранов
- **Контекстуальная информация**: Актуальные данные видны в нужное время и в нужном месте
- **Постепенное раскрытие**: Сложные функции открываются постепенно
- **Согласованные метафоры**: Стимпанк-алхимическая тема повсюду

### Визуальный язык
- **Цветовая палитра**: Тёплая латунь (#B8860B), медь (#B87333), кремовый (#F5F5DC), тёмно-коричневый (#3C2415)
- **Типографика**: Викторианский шрифт с засечками для заголовков, чистый беззасечечный для текста
- **Иконография**: Основанные на шестерёнках символы, алхимические символы, механические элементы
- **Интервалы**: Щедкие отступы для зон касания, чёткая визуальная иерархия

## Основные экраны интерфейса

### 1. Основной игровой экран (игровое поле)

#### Структура макета
┌──────────────────────────────────────────┐
│    Ресурс-бар │ Уведомления │ Меню       │
├──────────────────────────────────────────┤
│                                          │
│         ОСНОВНОЕ ИГРОВОЕ ПОЛЕ            │
│     (Сетка для размещения предметов)     │
│                                          │
├──────────────────────────────────────────┤
│ Инструменты │ Инвентарь │ Крафт │ Квесты │
└──────────────────────────────────────────┘


#### Элементы
- **Ресурс-бар**: Счётчики эфирных монет, паровых самоцветов, эссенции с иконками
- **Уведомления**: Обновления квестов, открытия достижений, анонсы событий
- **Основное игровое поле**: Сетка 8×10 для размещения и слияния предметов
- **Нижняя навигация**: Доступ к основным функциям с иконками в стиле шестерёнок

#### Шаблоны взаимодействия
- **Перетаскивание**: Основная механика слияния с тактильной обратной связью
- **Касание для выбора**: Выбор одного предмета с визуальным подсвечиванием
- **Двойное касание**: Быстрое отображение информации о предмете
- **Длительное касание**: Контекстное меню для продвинутых опций
- **Щипок для масштабирования**: Масштабирование игрового поля для доступности

#### Особенности

- **Выбор качества**: Опции для нормального, хорошего, отличного, идеального

3. Проверка требований к ингредиентам
4. Выбор уровня качества
5. Подтверждение крафта с отображением шанса успеха
6. Получение результатов с обновлением опыта и ресурсов

## Пользовательский поток

### Обучение нового игрока
1. **Приветственный экран**: Введение в тему и основную идею
2. **Создание персонажа**: Простые варианты настройки
3. **Интерактивное обучение**: Практическое слияние и крафт
4. **Первый квест**: Пошаговое завершение с наградой
5. **Экскурсия по лаборатории**: Введение в ключевые области интерфейса
6. **Свободное исследование**: Безопасный период экспериментов

### Цикл ежедневного вовлечения
1. **Награды за вход**: Немедленное позитивное подкрепление
2. **Просмотр квестов**: Проверка активных квестов и прогресса
3. **Управление ресурсами**: Сбор с генераторов, организация инвентаря
4. **Сессия крафта**: Завершение доступных рецептов
5. **Завершение квеста**: Сдача выполненных целей
6. **Планирование**: Установка целей для следующей сессии

### Оптимизация длительности сессии
- **Короткие сессии (5–15 мин)**: Быстрый сбор ресурсов, простой крафт
- **Средние сессии (15–45 мин)**: Завершение квестов, управление инвентарём
- **Длинные сессии (45+ мин)**: Сложный крафт, исследование, социальные функции

Этот UI/UX-дизайн гарантирует, что игра будет доступной, увлекательной и оптимизированной для мобильной игры, сохраняя при этом стимпанк-алхимическую тему во всех элементах интерфейса.
